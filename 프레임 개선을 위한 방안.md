#  imagezmq 라이브러리 사용
* imagezmq 라이브러리를 사용하여 pc(server) 와 client(raspberry pi)와 통신 
* 라즈베리파이는 카메라 영상을 pc로 전송해주고 pc는 전송받은 데이터로 openCV 작동
* imagezmq 라이브러리 사용 방법 : https://github.com/jeffbass/imagezmq
