# 라즈베리 파이 OS 설치 방법


### Headless Pi with WiFi
* Raspberry Pi Imager를 이용하여 Raspberry Pi OS를 마이크로 SD 카드에 저장
* SD 카드에 ssh와 wpa_supplicant.conf 쓰고 전원 ON
* RPi의 IP 주소를 알아낸 후 ssh로 접속
